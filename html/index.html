No Fuss Framework is a new opensource php mvc framework brought to you by Jarnix and Broutard.

It's hosted at github.

It's awesome and free.

Fast

  • Dynamic loading
  • Minimizing requires
  • Autoloading
  • Efficient routing system
  • Use multiple cache servers

Easy

  • Easy to learn
  • Web or cli applications
  • Internationalization
  • Built-in input filtering
  • Comes with a working example site

Powerful

  • Build multiple sites using the same models, controllers, and views
  • Deploy multiple versions (mobile, tablets...) using the same codebase
  • Maintained permanently since we use it every day in our apps !

Setup

Download

In this installation, we assume that your web documents are located in /var/www. Change accordingly to your configuration. 
cd /var/www
Download the latest version from github. 
git clone git://github.com/jarnix/nofuss.git
You will have two directories inside the "nofuss" directory : demosite and library. The library contains the classes of No Fuss Framework and the demosite is a basic demonstration site.

First test

Jump in the demosite/html directory. 
cd /var/www/nofuss/demosite/html
Run this command to use the cli version of the demosite : 
php ./index.php -a test/index/index
You should see something like "Hello world from cli !". Congratulations.
Of course the No Fuss Framework is made for powering websites but also works in the cli environment (you can use it for crons, long scripts, etc).

Setting up the demo website

We advise you to start with the demosite to understand how the framework works and hack your first web pages.

If your webserver is Apache, here is an example configuration : 
<VirtualHost 127.0.0.1>
	ServerName demo.nofussframework.com
	DocumentRoot /var/www/nofuss/demosite/html
	<Directory />
		Options FollowSymLinks
		AllowOverride All
		Order allow,deny
		Allow from all
	</Directory>
	ErrorLog /var/log/apache2/error-nofuss-demosite.log
</VirtualHost>
If you're lost, please refer to the Apache documentation to setup your website.

Now point your browser to http://demo.nofussframework.com. You should see something like "Hello world via PHP".

Default setup

No Fuss Framework is designed to be used as a library of functions to setup multiple sites, using common classes, and even models and controllers.
